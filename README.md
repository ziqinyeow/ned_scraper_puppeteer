# Prerequisite

Make sure you install make (able to run make file), if not use node cli

```
npm i
```

or

```
yarn
```

# Getting Started

```
make run <url> <url> <url> ...
```

or

```
node index.js <url> <url> <url> ...
```

or 

Modify test.js url_list, then run `make`, see the [Makefile](Makefile)
